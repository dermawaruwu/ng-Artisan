## ng-Artisan CMS

ng-Artisan is a content management system that build from Angular.js, Laravel, Bootstrap & Material Design Lite.
This open source project is belong to Iris Team.

## Installation

1. Composer install
2. setting your .env file
3. php artisan migrate
4. enjoy :)
